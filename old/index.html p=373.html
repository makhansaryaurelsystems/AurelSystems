Advanced Technologies Overview

CADSIM Plus includes a number of unique technologies that will enhance your process modeling experience:

Stream-based equations

By using its stream-based architecture, CADSIM Plus's FlexSpec™ technology allows you a high degree of flexibility in specifying your process simulation problem. You can equate one value to another, ratio, or calculate values for any unit module, or stream parameter. If you need even more complexity, you can use a series of equations, or cascading equations.

Values can be filtered, transformed, or passed through PID, or Fuzzy Logic controllers. Different values can even be selected for differing process conditions. CADSIM Plus specifications appear graphically on your flowsheet model so that you can survey and adjust these relationships quickly and easily. There are no values hidden inside blocks.

Click here for more information about why stream-based equations are important to you

CAD-based drawings

Living Drawings™ technology allows you to create a simulation model by sketching a process flowsheet. Anything that you can draw can be interpreted into a simulation by CADSIM Plus.

Click here for more information about how Living Drawings™ help you visualize and interact with your data

Create your own equipment modules

An Open Simulation Systems Architecture™ (OSSA™) was developed by Aurel to make CADSIM Plus an open simulation interface platform. CADSIM Plus can host different simulator "personalities". It is extensible. Users who are familiar with computer code development can purchase our module template and create their own custom process modules. In addition, our standard library source code is available for purchase.

Click here for more information about how our open architecture works

Faster runtime performance

CADSIM Plus's SmartCalc™ accelerates both simulation calculation and results display. SmartCalc manages flowsheet calculations to speed up runtime performance. SmartCalc-enhanced calculation times can be an order-of-magnitude faster than what is possible with any other process simulator.

SmartCalc gives CADSIM Plus the ability to run plant-wide dynamic simulations at 1000's of times faster than real time on entry-level PC computers. Our proprietary calculation 'by exception' system improves simulation performance on all computer hardware.

Click here to find out how SmartCalc improves runtime simulation performance

Flexible model interpretation

CADSIM Plus uses a hierarchy of process objects and a rules-based selection process to choose the most appropriate modules for any given simulation at runtime. SmartLink™ allows CADSIM Plus to host many differing simulation 'personalities'.

Advanced drawing parts

CADSIM Plus SmartParts™ automatically align themselves with the orientation of a pipe line, and may be set up to remove line segments, when they insert themselves into a line. SmartPart text automatically re-orients itself relative to the bottom of the drawing, in order to maintain drawing readability.

Select your level of model fidelity

Information re-scoping, promotion, and hiding allow the user to choose the level of fidelity needed for any area of the simulation model. This offers maximum model flexibility with a minimum amount of modeling time spent.

Click here to learn how CADSIM Plus allows you to choose the right level of modeling detail

Non-mass components

User-defined Quality or 'Special' Parameters allow tracking of components, which do not participate in normal mass, energy, and momentum balances, such as dollars, pH, viscosity, strength, and H+.

Automated controller tuning

Built-in wizards such as the AutoTune controller tuning and the user-defined chemical Compound Editor automate these complex tasks.

Data reconciliation and product tracking

Advanced online capabilities such as Dynamic Data Reconciliation (DDR), Product Tracking and Energy Tracking can be used to shadow live processes for various purposes, including corrected and enhanced process data collection and realtime process optimization and control.

Connectivity
  • DDE (Dynamic Data Change) - Can link to other Windows programs running on your PC. CADSIM Plus can act as both a DDE server and client.
  • COM (Common Object Model) - Can link to other Windows programs running on your PC. CADSIM Plus can act as both a COM server and client. Provides the ability to control CADSIM Plus simulation mode from another program, such as Excel.
  • OPC (OLE for Process Control - This optional library allows OPC connectivity to process control devices and software on your Local Area Network (LAN)
